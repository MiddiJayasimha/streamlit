# Skin Disease Classification

Complete Deep Learning Project for End-to-End AI-Powered Medical Image Analysis.

## Project Overview
This project implements an end-to-end deep learning system for automated skin disease classification. Leveraging transfer learning with the **EfficientNetB0** architecture, it classifies dermatological conditions across 38 different disease categories with high expected accuracy.

## Folder Structure
```

├── data/
│   └── balanced_dataset/          <- YOU MUST PLACE YOUR DATASET HERE
│       └── balanced_dataset/

```

## System Requirements
* Python 3.12 or higher
* CUDA-capable GPU (highly recommended for training)
* 16GB RAM (minimum 8GB)
* 10GB free disk space

## Installation Guide

1. Clone or navigate to this repository directory.
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux/Mac
   source venv/bin/activate
   ```
3. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```
4. Place your raw dataset in the `data/balanced_dataset/balanced_dataset/` folder so that each class is a sub-folder containing images.

## Usage Guide

Run scripts from the `scripts/` directory to ensure relative paths work correctly.

### 1. Data Analysis (EDA)
Generate dataset statistics, imbalance ratios, and distribution plots.
```bash
cd scripts
python skin_disease_eda_standalone.py
cd ..
```

### 2. Model Training
Train the complete EfficientNetB0 model. Will automatically save checkpoints, logs, and TFLite models.
```bash
cd scripts
python model_training_part1.py
cd ..
```

### 3. Model Evaluation
Compute accuracy, Cohen's Kappa, precision, recall, and generate confusion matrices.
```bash
cd scripts
python model_evaluation_part2.py
cd ..
```

### 4. Application Deployment
Launch the interactive web application used for clinical deployments.
```bash
streamlit run app/streamlit_app.py
```
